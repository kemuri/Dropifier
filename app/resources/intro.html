Dropifier is a tiny macOS app, that opens your Dropbox links directly in Finder.

  • Drag a file

    Simply drag and drop any file or folder from your Dropbox directory to the Dropifier icon in your menubar.

  • Paste the link

    Dropfier will create the link, and copy it to your clipboard automatically.

  • Enjoy

    Clicking on a Dropifier link will now open the given file or folder directly in Finder!
